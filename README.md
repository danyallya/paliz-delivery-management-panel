# Paliz Delivery Management Panel

A fully custom-built, comprehensive delivery management system for **Paliz**, designed to manage couriers, drivers, scheduling, and customer interactions. Built from scratch using **HTML5**, **CSS3**, and **jQuery** with a unique UI design and no reliance on pre-made templates or frameworks.

## 🚀 Overview
This is a large-scale internal dashboard/panel built for **Paliz**, a delivery and logistics company in Kerman, Iran. The system includes modules for managing:

- Couriers (Pik)
- Drivers
- Scheduling & Reservations
- Customer records
- Marketing agents
- Contracts
- Violations tracking
- Chat interface
- And more...

The panel supports all user roles including admin, driver, courier, and customer service representatives.

## 📋 Pages Included
- `index.html` – Dashboard
- `contact-us.html` – Contact page
- `marketing-panel.html` – For marketing agents
- `contract-signing.html` – Contract creation and signing
- `register-customer.html` – Customer registration
- `driver-info-entry.html` – Driver data entry
- `view-drivers.html` – Driver list for management
- `driver-details.html` – Driver profile
- `register-service.html` – Service registration
- `signed-contracts.html` – List of signed contracts
- `scheduling-table.html` – Time management and reservations
- `courier-list.html` – Courier (Pik) listing
- `chat-page.html` – Internal chat interface
- `violations.html` – Tracking violations

## 💻 Technologies Used
- **HTML5** – Semantic markup and accessibility
- **CSS3** – Responsive layout, animations, Flexbox/Grid
- **jQuery** – Dynamic interactions and DOM manipulation

> ⚠️ Note: This project was originally developed in **2018**, so JavaScript/jQuery files may require updates to avoid conflicts with modern practices. Please ensure compatibility when integrating into newer environments.
